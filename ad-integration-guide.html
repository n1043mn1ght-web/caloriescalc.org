Инструкция: внедрение рекламных блоков AdSense

Сайт: tdee-calculator-dad.pages.dev · Файл: ad-layout.css уже готов

Важно: Все изменения вносятся в 5 файлов — index.html, bmr-calculator.html, macro-calculator.html, protein-calculator.html, bmi-calculator.html. Принцип одинаковый для каждого. Файл about.html тоже можно обновить, но трафик там меньше.

Шаг 1 — Подключить ad-layout.css в <head> каждой страницы

Добавить одну строку в <head> каждого HTML-файла, после тега </style>:

<link rel="stylesheet" href="/ad-layout.css">

Шаг 2 — Обернуть <main> в .page-wrapper

Найти в каждом файле тег <nav>. Сразу после закрывающего </nav> добавить открывающий div, а после закрывающего </main> — закрывающий:

<!-- ДО (было): -->
</nav>

<main>
  ...контент...
</main>

<footer>


<!-- ПОСЛЕ (стало): -->
</nav>

<div class="page-wrapper">

  <aside class="ad-sidebar-left" aria-label="Advertisement">
    <div class="ad-slot ad-slot--skyscraper ad-slot--placeholder">
      <!-- AdSense код вставить сюда когда одобрят -->
      <!-- ins class="adsbygoogle" ... -->
    </div>
  </aside>

  <main>
    ...контент без изменений...
  </main>

  <aside class="ad-sidebar-right" aria-label="Advertisement">
    <div class="ad-slot ad-slot--skyscraper ad-slot--placeholder">
      <!-- AdSense код вставить сюда когда одобрят -->
    </div>
    <div class="ad-slot ad-slot--placeholder" style="min-height:250px;margin-top:16px">
      <!-- Второй блок справа 160x250 -->
    </div>
  </aside>

</div><!-- /.page-wrapper -->

<footer>

Шаг 3 — Добавить инлайн-блок после результатов калькулятора

Найти в коде каждого файла закрывающий тег блока результатов — обычно </div><!-- /results --> или </div><!-- /calc-card -->. Сразу после него вставить:

<!-- Рекламный блок после калькулятора (высокое внимание) -->
<div class="ad-slot ad-slot--inline ad-slot--placeholder"
     aria-label="Advertisement">
  <!-- AdSense код вставить сюда когда одобрят -->
  <!-- ins class="adsbygoogle" style="display:block" ... -->
</div>
Важно про плейсхолдер: Класс ad-slot--placeholder добавляет серый пунктирный прямоугольник — только для разработки, чтобы видеть где будет реклама. Когда вставишь реальный код AdSense — этот класс убрать, иначе серый фон будет поверх рекламы.

Шаг 5 — Когда придёт одобрение AdSense

Заменить содержимое каждого .ad-slot на код из панели AdSense. Убрать класс ad-slot--placeholder. Пример для скайскрепера:

<div class="ad-slot ad-slot--skyscraper">
  <ins class="adsbygoogle"
       style="display:block;width:160px;height:600px"
       data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"
       data-ad-slot="XXXXXXXXXX"></ins>
  <script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
</div>

Для инлайн-блока (адаптивный):

<div class="ad-slot ad-slot--inline">
  <ins class="adsbygoogle"
       style="display:block"
       data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"
       data-ad-slot="XXXXXXXXXX"
       data-ad-format="auto"
       data-full-width-responsive="true"></ins>
  <script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
</div>

Где добавить скрипт AdSense (один раз — в <head> каждой страницы)

<!-- AdSense скрипт — добавить в <head> когда придёт одобрение -->
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
     crossorigin="anonymous"></script>
Совет по AdSense Auto Ads: После одобрения можно включить Auto Ads в панели AdSense — Google сам расставит блоки по всему сайту дополнительно к ручным. Это не отменяет ручные блоки, а дополняет. На старте проще включить Auto Ads и не думать о ручной расстановке.
